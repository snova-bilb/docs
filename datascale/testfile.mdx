Here’s an overview of the doc pages:

* [**Get started with DataScale administration**](https://docs.sambanova.ai/sysadmin/latest/hw-admin-get-started.html). Learn how to get started and where to find more information.

* [**Network administration**](https://docs.sambanova.ai/sysadmin/latest/network-admin.html). Learn from third-party documentation and look at some examples.

* [**DataScale power management**](https://docs.sambanova.ai/sysadmin/latest/manage-power.html). Familiarize yourself with warnings and understand the boot process, power on process, and graceful shutdown.

* [**Host module OS administration**](https://docs.sambanova.ai/sysadmin/latest/host-module-admin.html). Learn about common tasks performed on the host module, which supports RedHat and Ubuntu.

* [**BMC administration**](https://docs.sambanova.ai/sysadmin/latest/bmc-admin.html). Learn how to update and how to recover the BMC (base model controller) firmware.

* [**RDU module administration**](https://docs.sambanova.ai/sysadmin/latest/hw-admin-rdu.html). Learn about RDU module tasks such as changing the root password, generating an API token, and configuring the network and hostname.

* [**Monitor and debug DataScale**](https://docs.sambanova.ai/sysadmin/latest/hw-monitor-debug.html). Learn about different tools for monitoring and troubleshooting, and how to collect diagnostic information for Customer Support.

* [**Back up and restore components**](https://docs.sambanova.ai/sysadmin/latest/hw-backup-restore.html). Find links to KB articles for recovering different components of the DataScale rack.
